# yt-dlp-music

Search YouTube for tracks/albums and download authorized audio with `yt-dlp`.

## What It Does

1. Verifies `yt-dlp` and `ffmpeg` availability.
2. Searches YouTube with compact candidate output.
3. Previews ambiguous track, album, and playlist matches before downloading.
4. Downloads selected audio into `~/Downloads/yt-dlp-music/` by default.
5. Embeds metadata and thumbnails when possible.
6. Reports created files, skipped files, metadata gaps, and errors.

## Invoke

Triggers include:

- "search YouTube for this album"
- "download this track with yt-dlp"
- "download this album"
- "rip audio from YouTube"
- "find this song on YouTube"
- "yt-dlp music"
